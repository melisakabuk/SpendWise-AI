# SpendWise AI: The Autonomous Wealth Guardian

## 1. Project Vision
SpendWise AI is an intelligent financial assistant designed to manage digital subscriptions and optimize spending. In an era of "subscription fatigue," it acts as a dedicated guardian for the user's finances.

## 2. MVP Scope (Minimum Viable Product)
The MVP focuses on three core pillars:
* **Subscription Detection:** Identifying recurring payments from transaction data.
* **Predictive Alerts:** Forecasting end-of-month balance based on "burn rate."
* **Ghost Card Manager:** Providing virtual cards for safe "free trial" sign-ups.

## 3. Product Requirements Document (PRD)

### A. Functional Requirements
| ID | Feature | Description | Priority |
|:---|:---|:---|:---|
| FR.01 | AI Detection | Automatically flag recurring bills in CSV/JSON uploads. | High |
| FR.02 | Burn Rate UI | A dashboard showing upcoming payments vs. current balance. | High |
| FR.03 | Negotiation Bot | Generate automated email scripts to request discounts from providers. | Medium |

### B. User Stories
* **As a user**, I want to be notified 2 days before a "free trial" expires so I can cancel it.
* **As a user**, I want to see a summary of how much I saved through AI negotiations this year.

## 4. Success Metrics (KPIs)
* **Total Savings:** The amount of money saved per user through cancellations.
* **Retention:** Monthly active users managing their "Ghost Cards."

## 5. Future Roadmap
* Open Banking (PSD2) API integration for real-time bank feeds.
* Automated subscription cancellation with one click.
